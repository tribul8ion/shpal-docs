## Введение

Команды для принтера Godex G530U можно вводить двумя способами: через свойства принтера в Windows или через программу GoDoctor.

## Основные команды

### Размеры и позиционирование

<Accordions>

  <Accordion id="label-size" title="Размер этикетки">

**Ширина этикетки:**
```
^W80
```
Где **80** — ширина в миллиметрах.

**Высота этикетки:**
```
^Q60
```
Где **60** — высота в миллиметрах.

**Пример для этикетки 80×60 мм:**
```
^W80
^Q60
```

  </Accordion>

  <Accordion id="position" title="Позиционирование содержимого">

**Положение по горизонтали:**
```
^R0
```
Где **0** — положение по горизонтали (0 = по центру).

**Положение по вертикали:**
```
~Q+0
```
Где **+0** — положение по вертикали (0 = по центру).

<Callout type="info">
  Отрицательные значения сдвигают содержимое влево/вверх, положительные — вправо/вниз.
</Callout>

**Примеры:**
```
^R-5   # Сдвиг влево на 5 мм
^R+10  # Сдвиг вправо на 10 мм
~Q-3   # Сдвиг вверх на 3 мм
~Q+7   # Сдвиг вниз на 7 мм
```

  </Accordion>

  <Accordion id="stop-position" title="Положение остановки">

**Положение остановки после печати:**
```
^E20
```
Где **20** — сколько миллиметров будет торчать следующая этикетка из принтера (возможно удобно для отрыва).

**Рекомендуемые значения:**

| Конфигурация | Значение | Описание |
|--------------|----------|----------|
| Без отделителя/резака | 17-18 мм | Стандартное значение |
| С отделителем | 9-10 мм | Для автоматического отделения |
| С резаком | 29-30 мм | Для автоматической резки |

**Примеры:**
```
^E17   # Стандартная позиция
^E10   # С отделителем
^E30   # С резаком
```

  </Accordion>

</Accordions>

### Параметры печати

<Accordions>

  <Accordion id="speed-density" title="Скорость и плотность">

**Скорость печати:**
```
^S4
```
Где **4** — скорость печати (диапазон обычно 2-6).

| Значение | Скорость | Применение |
|----------|----------|------------|
| 2 | Очень медленная | Максимальное качество |
| 3 | Медленная | Высокое качество |
| 4 | Средняя | Стандартная печать |
| 5 | Быстрая | Черновая печать |
| 6 | Очень быстрая | Минимальное качество |

**Плотность печати:**
```
^H13
```
Где **13** — плотность печати (обычно 1-20).

| Значение | Плотность | Применение |
|----------|-----------|------------|
| 5-8 | Низкая | Тонкие этикетки |
| 10-13 | Средняя | Стандартные этикетки |
| 15-20 | Высокая | Плотные материалы |

**Пример настройки:**
```
^S4    # Средняя скорость
^H13   # Средняя плотность
```

  </Accordion>

  <Accordion id="copies" title="Количество копий">

**Число копий:**
```
^P1
```
Где **1** — количество копий для печати.

**Примеры:**
```
^P1    # Одна копия
^P5    # Пять копий
^P10   # Десять копий
```

  </Accordion>

</Accordions>

### Ориентация и переворот

<Accordions>

  <Accordion id="rotation" title="Поворот этикетки">

**Переворот этикетки:**
```
~R255
```
Переворачивает изображение на 180°.

**Поворот при печати (для бейджей из Word):**
```
^XSET,ROTATION,n
```

Где **n**:
- **0** — поворот 0° (без поворота)
- **1** — поворот 90°
- **2** — поворот 180°
- **3** — поворот 270°

**Примеры:**
```
^XSET,ROTATION,0   # Без поворота
^XSET,ROTATION,1   # Поворот на 90°
^XSET,ROTATION,2   # Поворот на 180°
^XSET,ROTATION,3   # Поворот на 270°
```

<Callout type="info">
  Используйте команду поворота при печати бейджей из Word для правильной ориентации.
</Callout>

  </Accordion>

</Accordions>

### Дополнительные функции

<Accordions>

  <Accordion id="cutter" title="Резак">

**Настройка резака:**
```
^XSETCUT,DOUBLECUT,0
```

Включает/выключает функцию двойной резки.

**Параметры:**
- **0** — резак выключен
- **1** — резак включен

**Примеры:**
```
^XSETCUT,DOUBLECUT,0   # Резак выключен
^XSETCUT,DOUBLECUT,1   # Резак включен
```

<Callout type="warn">
  Если резак установлен, установите положение остановки **^E30** (29-30 мм).
</Callout>

  </Accordion>

  <Accordion id="test-page" title="Тестовая страница">

**Печать тестовой страницы:**
```
~V
```

Печатает тестовую страницу с текущими настройками принтера.

<Callout type="info">
  Используйте эту команду для проверки всех параметров принтера после настройки.
</Callout>

  </Accordion>

</Accordions>

## Примеры конфигураций

### Стандартная этикетка 80×60 мм

```
^W80          # Ширина 80 мм
^Q60          # Высота 60 мм
^S4           # Средняя скорость
^H13          # Средняя плотность
^E17          # Стандартная позиция остановки
^R0           # Центрирование по горизонтали
~Q+0          # Центрирование по вертикали
^P1           # Одна копия
```

### Этикетка с отделителем

```
^W80          # Ширина 80 мм
^Q60          # Высота 60 мм
^S4           # Средняя скорость
^H13          # Средняя плотность
^E10          # Позиция для отделителя
^R0           # Центрирование по горизонтали
~Q+0          # Центрирование по вертикали
^P1           # Одна копия
```

### Этикетка с резаком

```
^W80                    # Ширина 80 мм
^Q60                    # Высота 60 мм
^S4                     # Средняя скорость
^H13                    # Средняя плотность
^E30                    # Позиция для резака
^XSETCUT,DOUBLECUT,1    # Резак включен
^R0                     # Центрирование по горизонтали
~Q+0                    # Центрирование по вертикали
^P1                     # Одна копия
```

### Бейдж из Word (книжная ориентация)

```
^W80                  # Ширина 80 мм
^Q60                  # Высота 60 мм
^S3                   # Медленная скорость (качество)
^H15                  # Повышенная плотность
^E17                  # Стандартная позиция
^XSET,ROTATION,0      # Без поворота
^R0                   # Центрирование
~Q+0                  # Центрирование
^P1                   # Одна копия
```

## Быстрая настройка

### Шаблон для копирования

Скопируйте и вставьте нужный набор команд:

```
# Базовая конфигурация 80×60
^W80
^Q60
^S4
^H13
^E17
^R0
~Q+0
^P1
~V
```

<Callout type="info">
  Команда **~V** в конце распечатает тестовую страницу для проверки настроек.
</Callout>

## Устранение проблем

<Accordions>

  <Accordion id="commands-not-work" title="Команды не применяются">

**Причина:** Неправильный синтаксис или команды не отправлены на принтер.

**Решение:**
1. Проверьте правильность написания команды
2. Убедитесь, что команда отправлена (кнопка Apply/Применить)
3. Перезагрузите принтер
4. Попробуйте ввести команды через GoDoctor
5. Напечатайте тестовую страницу `~V` для проверки

  </Accordion>

  <Accordion id="wrong-position" title="Неправильное позиционирование">

**Причина:** Неправильные значения команд `^R` или `~Q`.

**Решение:**
1. Начните с нулевых значений: `^R0` и `~Q+0`
2. Постепенно изменяйте значения на 1-2 мм
3. Печатайте тестовую этикетку после каждого изменения
4. Положительные значения — вправо/вниз, отрицательные — влево/вверх

  </Accordion>

  <Accordion id="cutter-issues" title="Проблемы с резаком">

**Причина:** Неправильная позиция остановки или настройки резака.

**Решение:**
1. Убедитесь, что резак включен: `^XSETCUT,DOUBLECUT,1`
2. Установите правильную позицию: `^E30` (29-30 мм)
3. Проверьте, что резак физически установлен и работает
4. Выполните калибровку принтера

  </Accordion>

</Accordions>
